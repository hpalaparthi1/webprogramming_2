Description:

The fifteen puzzle is a simple game first invented around 1874. The game contains 15 blocks in a 4Ã—4 grid, with each block labeled with an integer from 1 to 15. The goal of this puzzle is to arrange the block sequentially from 1 to 15 with the empty block (which would be numbered 16 if it were there) in the bottom right. In modern times the numbers have been replaced by images, which, when the game has been solved, form a complete picture. To start, the blocks are placed randomly in the grid. It should be noted, however, that certain initial conditions are unsolvable. HTML, CSS and JavaScript.
